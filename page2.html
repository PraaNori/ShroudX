Text in Text Steganogaphy

Browse through the below techniques

Steganography using Zero Width Characters can encode upto 250 words. It converts the secret text into zero width characters like word joiner, zero-width no-break space, etc and stores it in the cover text.

Steganogaphy using Caeser Cipher method converts the existing text into it's equivalent letter using a number. It decodes back using the same.
